SnailType

Class project proposal

Product Name: SnailType

  • Team Members: King
  • Application Overview: load a bank of random words and type out as many as you can in a set time
  • Technology Overview: HTML, CSS, JS, GitHub
  • Proposal Last Updated: 4/23/26

Gameplay Features

  • Grey out letters that aren’t typed
  • Red boxes for missed or wrong letters
  • After the time is up, show WPM and accuracy percentage

Future Feature List

  • Try and have a leaderboard if possible
  • Basic Technical Features: design a visually pleasing website
  • Advanced Technical Features: use a text box to save local scores, change greyed letters color based on input
